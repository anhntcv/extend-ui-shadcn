## Useful behaviors

- Actual and expected values displayed side by side for each schema field.
- Input controls selected from the JSON schema property type.
- Per-field undo and Set to NULL actions.
- Form and JSON diff views for the same reviewed output state.

## Common slots

| Slot       | Purpose                                               |
| ---------- | ----------------------------------------------------- |
| `field`    | A schema property, extracted value, and review value. |
| `actual`   | Read-only machine output for comparison.              |
| `expected` | Editable reviewed output for correction.              |
| `diff`     | JSON-level actual versus expected comparison.         |
